Selectors

Selectors

Element Selector
p, div....etc
HTML tag name ကို တိုက်ရိုက်ခေါ်သုံးတာကို Element Selector လို့ခေါ်တယ်
p{
    font-size:20px;
}

Single Selector
တစ်ခုတည်း select လုပ်ထားတာကို ခေါ်ပါတယ်
p{
    color:red;
}

Group Selecors (p , p)
တူညီ တဲ့ style ရှိစေချင်တာကြောင့် အတူ တကွ select လုပ်ပေးတာကို Group Selector လို့ခေါ်ပါတယ်။
p, p {
    color:red;
}
h1, h2, h3, h4{
    color:burlywood;
}

ID selectors 
ID ကနေ select လုပ်။ # sign လေးက ID အတွက် မဖြစ်မနေ လိုအပ်
#car{
    color:crimson;
}
#car1 , #car2{
    color:black;
}

Class Selectors
Class ကနေ select လုပ်။ . sign လေးက Class အတွက် မဖြစ်မနေ လိုအပ်
.ph {
    color:skyblue
}
.f20 {
    font-size: 20px;
}
.divone, .divtwo{
    color:darkgreen;
}
